## How to create a local repository:

* Ver el directorio actual: $ pwd
* Hacer el repo local: $ mkdir nombre
* Cambiar al directorio del repo: $ cd nombre
* Inicializar el repo: $ git init
* Seleccionar los archivos añadidos al repo local: $ git add --all
* Asignar los archivos: $ git commit -m 'initial commit'

## How to add your local repository on GitHub:

* Crear un repositorio en GitHub
* Añadir la dirección remota del repositorio: $ git remote add origin https://github.com/github-username/repo-name.git
* Subir los archivos locales: $ git push origin master
* Escribir usuario y contraseña
* Actualizar GitHub en el navegador y listo
